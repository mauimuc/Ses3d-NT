checks_test.f90

Example program which tests procedures of the module checks_mod The following checks are carried out:

  • Check if the source file exists
  • Check existence of multiple files
  • Check latitude
  • Check longitude
  • Check depth

Compile with simply using the Makefile: 

 make checks_test

Run the program by executing: 

 ./checks_test
